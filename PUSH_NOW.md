# 🚀 Push to GitHub NOW - Simple Steps

## Quick Commands (Copy & Paste)

Open your terminal in the project root and run these commands:

```bash
# Step 1: Initialize git
git init

# Step 2: Add all files
git add .

# Step 3: Commit
git commit -m "Initial commit: ForecastIQ - ML Forecasting Platform"

# Step 4: Set main branch
git branch -M main

# Step 5: Add remote
git remote add origin https://github.com/satvik-sharma-05/ForecastIQ.git

# Step 6: Push to GitHub
git push -u origin main
```

## Or Use the Script

### Windows (PowerShell or CMD)
```bash
push_to_github.bat
```

### Linux/Mac
```bash
chmod +x push_to_github.sh
./push_to_github.sh
```
